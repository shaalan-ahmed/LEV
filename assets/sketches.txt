user1@user:~ safepwd help
    store your password safely with safepwd

    <options>
        help -> look at options

        new -> register new user
            usage -> new <username> <password>

        login -> login to existing user
            usage -> login <username> <password>

        delete -> delete existing user
            usage -> delete <username> <password>

        exit -> quit safepwd

user1@user:~ safepwd -n john "john's password"
user1@user:~ safepwd -u john "john's password"

safepwd@john: help
    <options>
        list -> list entries
            usage -> list

        add -> add new entry
            usage -> add

        rm -> remove existing entry
            usage -> rm <EntryName>

        search -> prints entry
            usage -> search <EntryName>

        edit -> edit existing entry
            usage -> edit <EntryName>

        exit -> quit safepwd
