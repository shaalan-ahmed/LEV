user1@user:~ lev help
    store your password safely with Locally Encrypted Vault (LEV)

    <options>
        help: look at options 

        new: register new user
            usage: new <username> <password>

        login: login to existing user
            usage: login <username> <password>

        delete: delete existing user
            usage: delete <username> <password>

        exit: quit lev

user1@user:~ lev new john "john's password"
user1@user:~ lev login john "john's password"

lev@john: help
    <options>
        list: list entries
            usage: list

        add: add new entry
            usage: add <name> <url> <password>

        rm: remove existing entry
            usage: rm <name>

        search: prints entries containing <query> using fuzzy search
            usage: search <query>

        edit: edit existing entry
            usage: edit <name> <newname> <newurl> <newpassword>

        copy: copies the password of an entry
            usage: copy <name>

        gen: generates a secure password and copies it to user's clipboard
            usage: gen

        exit: quit lev
